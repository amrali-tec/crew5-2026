# Crew5 Agency Website

Creative agency website built with **Next.js 14**, **Tailwind CSS**, and **TypeScript**.

## Features

- Fully responsive layout
- Hero sections for homepage and pages
- Services, Projects, Blog, Testimonials, Stats sections
- Partners carousel with infinite autoplay
- Contact form and WhatsApp integration
- Modern animations and hover effects
- Clean, modular components

## Development
npm run dev

##Production
npm run build
npm run start

```bash
git clone https://github.com/yourusername/crew5-website.git
